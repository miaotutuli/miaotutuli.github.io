man - manual - 查看手册

pwd - print working directory - 打印工作目录

cd - change directory
~ 绝对路径：以/开头
~ 相对路径：以当前路径作为基准

ls - list directory contents
~ -l：长格式
~ -a：查看所有
~ -R：递归式查看

touch - 创建一个空文件或修改文件的最后访问时间

mkdir - make directory - 创建文件夹
~ -p：在需要的时候创建父文件夹

rmdir - remove empty directory - 删除空文件夹

rm - remove - 删除文件或文件夹
~ -i：interactive - 交互式删除（询问）
~ -f：force - 强制删除（不询问）
~ -r：recursive - 递归式删除（删文件夹）

cp - copy - 拷贝文件或文件夹
~ -r：递归式拷贝（拷贝文件夹）

mv - move - 移动文件或文件夹

cat - concatenate - 连接多个文件打印文件内容
~ -n：显示行号

tac - 反向查看文件

head - 查看文件的开头

tail - 查看文件的末尾

more / less - 分页查看文件

wc - word count - 统计字符、单词、行数

file - 查看文件属性

od - 查看二进制文件的数据

Tab：命令或路径补全
Ctrl+w：删除一个单词
Ctrl+u：删除整行
Ctrl+a：光标移到行首
Ctrl+e：光标移到行尾

Linux安装软件：

  1. 包管理工具 - yum / rpm
    • 查找：yum search
    • 安装：yum install / rpm -ivh <rpm包>
    • 卸载：yum erase / yum remove / rpm -e
    • 查看：yum info / rpm -qa | grep
    • 已安装：yum list installed
  2. 源代码构建安装 - 下载源代码编译安装
  3. 下载解压缩配置环境变量

Linux下安装Python3(源代码构建安装)

  1. 下载
    wget https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tar.xz

  2. 解压缩和解归档
    xz -d Python-3.7.4.tar.xz
    tar -xvf Python-3.7.4.tar

  3. 切换到Python源代码文件夹
    cd Python-3.7.4

  4. 补充安装依赖项（C语言的库）
    yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel db4-devel libpcap-devel xz-devel libffi-devel

  5. 构建前的配置（生成构建文件Makefile）
    ./configure –prefix=/usr/local/python37 –enable-optimizations

  6. 执行构建安装
    make && make install

  7. 配置环境变量 vim ~/.bash_profile (用户环境变量)

    更改：export PATH=$PATH:/usr/local/python37/bin
